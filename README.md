# QtDocument_CN

本项目为翻译 Qt 官方文档，内容基于 Qt 最新发布版本 5.15。

## 关于社区

### 参与者公约
参与相关讨论时，请遵守[参与者公约](CODE_OF_CONDUCT.md)。

### 加入社区

此外，我们组建了[QtDocumentCN-社区组]微信群，可以添加[@CryFeiFei](https://github.com/CryFeiFei) 的微信`Perfect_Cheung`，备注QtDoc，待邀请入群后在群里详细聊。
